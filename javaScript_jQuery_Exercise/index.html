  5. Determine the product of all the numbers

Easy Factorial

Write a program that calculates the factorial of a number. The factorial of a non-negative integer n is the product of all positive integers less than or equal to n. (i.e. 5! = 5 x 4 x 3 x 2 x 1 = 120).

Insert factorial number: 

                                function GetFactorial() {
                                    var num = parseInt($("#n1").val());
                                    var ans = 1;
                                    while (num > 1) {
                                        ans = ans * num--; 
                                        //post decrement - very useful
                                    }
                                    $("#fact-result").html(ans + " is the factorial number.");
                                }

1 - 100 FizzBuzz

Write a program that accepts two numbers between 1 and 100 from the user. The program then iterates through all numbers from 1 to 100, printing numbers in ascending order to the screen. For each number that is a multiple of the first number input by the user, print "Fizz" instead of the number. For each number that is a multiple of the second number input by the user, print "Buzz" instead of the number. For numbers that are multiples of both numbers, print "FizzBuzz" instead of the number.

                                    function fizzBuzz() {
                                        var str = "";
                                        for (i = 1; i <= 100; i++) {
                                            if ((i % 3 == 0) && (i % 5 == 0)) {
                                                str += i + " FizzBuzz ";
                                            } else if ((i % 3 == 0)) {
                                                str += i + " Fizz ";
                                            } else if ((i % 5 == 0)) {
                                                str += i + " Buzz ";
                                            } else {
                                                str += i + " ";
                                            }
                                        }
                                        $("#fizzbuzz-result").html(str);
                                    }

Simple Palindrome

Write a program that determines whether a word is a Palindrome (the same backward as it is forward – “radar” for example). Applies only for single words. (use same caps) Display the result to the user.

Palindrome check:

                                function palindrome() {
                                    outputWord = "";
                                    inputWord = document.getElementById("word").value;
                                    for (var i = inputWord.length; i >= 0; i--) {
                                        outputWord += inputWord.charAt(i);
                                    }
                                    if (inputWord === outputWord) {
                                        $("#pal-result").html(  inputWord + 
                                                            " is a Palindrome.");
                                    }
                                    else {
                                        $("#pal-result").html(  inputWord + 
                                                            " is not a Palindrome.");
                                    }
                                }
